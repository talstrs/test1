Source File Structure


CSS Source File structure

These are the css files that are loaded into templates in Head Section. 


            <!-- Font CSS -->
            <link rel="stylesheet" href="./assets/css/giulia-plain.css" />            

            <!-- Vendor CSS (Bootstrap & Icon Font) -->
            <link rel="stylesheet" href="./assets/css/bootstrap.min.css" />
            <link rel="stylesheet" href="./assets/css/lastudioicon.css" />
            <link rel="stylesheet" href="./assets/css/dlicon.css" />

            <!-- Plugins CSS (All Plugins Files) -->
            <link rel="stylesheet" href="./assets/css/swiper-bundle.min.css" />
            <link rel="stylesheet" href="./assets/css/glightbox.min.css" />
            <link rel="stylesheet" href="./assets/css/nice-select2.css" />

            <!-- Style CSS -->
            <link rel="stylesheet" href="./assets/css/style.css" />


Javascript Source File structure

These are the list of Javascript files that are loaded into templates in end of the Body Section. 

       
            <!-- JS Vendor, Plugins & Activation Script Files -->

            <!-- Bootstrap JS -->
            <script src="./assets/js/bootstrap.bundle.min.js"></script>

            <!-- Plugins JS -->
            <script src="./assets/js/swiper-bundle.min.js"></script>
            <script src="./assets/js/masonry.pkgd.min.js"></script>
            <script src="./assets/js/glightbox.min.js"></script>
            <script src="./assets/js/nice-select2.js"></script>

            <!-- Activation JS -->
            <script src="./assets/js/main.js"></script>
